# React + TypeScript + Vite

Comandos de interés:

```bash
npm install
# Instala todas las dependencias listadas en package.json

npm i -D daisyui@latest
# Instala DaisyUI (plugin de Tailwind CSS) como dependencia de desarrollo

npm i react-router
# Instala React Router para manejo de rutas en aplicaciones React

npm install @mui/material @emotion/react @emotion/styled @mui/icons-material
# Instala Material UI y sus dependencias para componentes de interfaz de usuario en React

npm install dayjs
# Instala Day.js para manipulación y formateo de fechas

npm install --save-dev --save-exact prettier
# Instala Prettier como dependencia de desarrollo para formateo de código

node --eval "fs.writeFileSync('.prettierrc','{}\n')"
# Crea un archivo de configuración vacío para Prettier (.prettierrc)

node --eval "fs.writeFileSync('.prettierignore','# Ignore artifacts:\nbuild\ncoverage\n')"
# Crea un archivo .prettierignore para ignorar ciertos archivos y carpetas al formatear con Prettier

npm install --save-dev eslint-config-prettier
# Instala eslint-config-prettier para desactivar reglas de ESLint que puedan entrar en conflicto con Prettier
```
